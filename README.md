## MorPhiC gene list selection tool

This tool was developed to facilitate the gene selection process for the MorPhiC project. It contains an annotated set of all protein coding genes (metadata used outline below) as well as interactive visualisations. The tools also allows users to view the genes currently selected by the data production centers to generate null alleles in cells.

*Metadata & resources*

Mouse knockouts:
- IMPC viability and phenotype screenings
- MGI viability screenings

Gene constraint metrics (cell lines):
- DepMap CERES scores
- H1-iCas9 Genetic Screens

Gene constraint metrics (population sequencing):
- gnomAD loss-of-function LOEUF
- Selection against heterozygous Loss of Function (rgcme)
- Selection against heterozygous Loss of Function (posterior)
- DOMINO score
- SCoNeS scores

Disease data:
- Online Mendelian Inheritance in Man (OMIM)
- Development disorder gene2phenotpye (DDG2P)

Other:
- Gene Ontology
- PANTHERdb protein families
- Reactome pathways

The application can be found at: https://whri-phenogenomics.shinyapps.io/morphic_gene_list/.
